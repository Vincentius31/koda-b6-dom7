# Rick and Morty Character Display and Search

Aplikasi ini merupakan sebuah penampilan character dari kartun Rick and Morty yang mengambil data dari API luar dan ditampilkan ke dalam halaman web. Aplikasi ini juga mampu untuk melakukan searching secara langsung hanya tinggal masukan nama karakter yang ingin dicari.

## Cara menjalankan app

1. Buka file index.html.
2. Lakukan pencarian karakter pada search bar
3. Hasil pencarian akan langsung muncul
